### Features
- Wallet :
    - send / receive
    - masternode status and type 
- Masternode Contract Interaction
    - create masternode
    - disable masternode
- Poseidon connector
    - view all payments status per node
- Remote server monitoring dashboard
- base framework build for the Pirl ecosystem
